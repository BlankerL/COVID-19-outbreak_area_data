# COVID-19-outbreak_area_data
## 数据源:![数据源](/tools/source.png)
***
**COVID-19疫情区域数据，特点：不采用非官方民间数据**

![地图](/tools/map.png)

### 字段说明

 > 行政区划下的疫情区域数量 [**City.csv**](/data/City.csv)
 >> 字段 | 含义
 >> -----|-----
 >> pName  | 省份名
 >> pId    | 省份编号
 >> pTotal | 省份总计疫情区域数量
 >> name   | 城市名
 >> id     | 城市编号
 >> lon    | 经度
 >> lat    | 纬度
 >> level  | 定位精度
 >> count  | 城市疫情区域数量
 >>
 >具体疫情地区信息 [**Pois.csv**](/data/Pois.csv)
 >
 >> 字段 | 含义
 >> -----|-----
 >> poiname | 疫情地区名
 >> lon     | 经度
 >> lat     | 纬度
 >> source  | 信息来源
 >>
